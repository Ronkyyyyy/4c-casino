# 🎰 四字熟語カジノ (Yojijukugo Casino)

**「知識」と「運」が交差する、漢字画数ベッティングゲーム。**

ひらがなで提示される四字熟語から、漢字に変換した際の「画数」を予想してBETするWebブラウザゲームです。
単純な知識クイズではありません。「最大画数」を狙うハイリスク・ハイリターンな心理戦と、重複（バッティング）による没収の恐怖があなたを待っています。

## 🎮 ゲーム概要

プレイヤーはディーラーから配られた「ひらがなカード」を見て、どの範囲が1つの漢字になるかを予想し、BETします。
見事的中すれば、その漢字の **「画数」** がそのままスコアになります。

### 特徴
- **リッチな演出**: 扉が開くギミック、カードがリールに吸い込まれるアニメーション、ドラムロール演出など、カジノの臨場感を再現。
- **スリリングなルール**: 画数が被ると0ptになる「バッティング没収」システムにより、単に画数が多い字を選ぶだけでは勝てない戦略性を実現。
- **レスポンシブ**: PC/スマホ両対応（推奨はPCブラウザ）。

## 📜 ルール

全10ラウンドを行い、最終的な獲得チップ（スコア）を競います。

1.  **出題**: 
    ランダムに選ばれた四字熟語が「ひらがな」の状態で配られます。
2.  **BETタイム**:
    制限時間（10秒）以内に、**「ここが1つの漢字だ」**と思うひらがなの範囲を選択します。
    * 例：「ききいっぱつ」→「き」を選択（危? 機?）
3.  **JUDGEMENT（判定）**:
    * **区切り正解**: 漢字の「画数」分のポイントを獲得。
    * **区切り不正解**: 手持ちポイントが**半減**（初回のみペナルティ免除）。
4.  **ボーナス & ペナルティ**:
    * **JACKPOT**: その熟語の中で「最大画数」の漢字を単独で当てれば、ボーナス **+3pt**。
    * **BAT OUT (没収)**: 選んだ漢字の画数が、他の漢字と**重複（バッティング）**していた場合、得点は **0pt** になります。
